Введение в Tailwind CSS
Ну что ж, наш разговор о стилизации в React будет неполным, если я не расскажу о Tailwind. Название переводится как «попутный ветер» (Tailwind). Tailwind CSS — это утилитарный CSS-фреймворк, который позволяет быстро создавать пользовательские интерфейсы, используя готовые классы.

В отличие от традиционных CSS-фреймворков типа Bootstrap, Tailwind не предоставляет готовых компонентов — всяких кнопочек, карточек, ссылок или аккордеонов. Вместо этого он предлагает низкоуровневые утилитарные классы для стилизации. Основное его преимущество состоит в том, что он позволяет быстро стилизовать элементы, не переключаясь между HTML и CSS файлами. То есть весь CSS вы пишете прямо в JSX, используя готовые классы.

Установка и настройка (версия 4.0)
Я записываю этот урок в марте 2025 года, и в Tailwind недавно произошли очень большие изменения: появилась четвертая версия, с которой мы и будем работать. На сегодняшний день это уже стабильный вариант, который практически не изменится в плане работоспособности.

По сравнению с предыдущими версиями, четвертая стала намного удобнее для установки. Если вы используете Vite (а мы работаем именно с ним), установка выполняется довольно просто. Я предварительно удалил из проекта все лишние компоненты и CSS-файлы, чтобы начать с чистого листа.

Установка: Вводим команду для установки Tailwind и плагина для Vite: npm install tailwindcss @tailwindcss/vite

Конфигурация Vite: В файле vite.config.js нужно добавить импорт Tailwind и подключить его в массив плагинов:

JavaScript

import tailwindcss from '@tailwindcss/vite'
// ... в plugins добавить tailwindcss()
Подключение CSS: Создаем основной файл стилей (например, main.css). В нем теперь достаточно одной строки: @import "tailwindcss";

Импорт: Остается только импортировать этот CSS-файл в наш основной файл main.jsx.

Начало работы и расширение IntelliSense
Чтобы работа была комфортной, обязательно установите расширение Tailwind CSS IntelliSense для VS Code. Оно дает всплывающие подсказки: когда вы пишете класс, расширение показывает, какому именно CSS-свойству он соответствует. Это очень помогает на этапе обучения.

Базовая стилизация (Текст, Блоки, Кнопки)
Начнем со стилизации заголовка H1. Используем атрибут className:

text-4xl — устанавливает размер шрифта.

font-bold — делает шрифт жирным.

text-blue-900 — задает цвет текста.

text-center — выравнивание по центру.

mt-20 — отступ сверху (margin-top). Одна единица в Tailwind по умолчанию равна 4 пикселям, то есть mt-20 — это 80px.

Для блоков (div):

bg-blue-500 — цвет фона.

w-30 — ширина.

rounded-full — делает блок круглым или сильно скругляет углы.

mx-auto — центрирование блочного элемента по горизонтали.

border-4 — толщина рамки.

Для кнопок и интерактивных элементов:

hover:bg-blue-400 — изменение фона при наведении.

w-1/3 — ширина элемента в процентах (в данном случае 33.3%).

transition-colors и duration-900 — плавный переход цвета в течение 900мс.

Чтобы строчный элемент (например, кнопка) начал слушаться отступов и ширины, можно добавить класс block.

Работа с Flexbox и Grid
Tailwind по умолчанию работает по принципу Mobile First (сначала мобильные).

Flexbox: Достаточно написать класс flex. По умолчанию элементы встают в ряд. С помощью префиксов можно управлять адаптивностью: например, flex-col md:flex-row означает, что на мобильных устройствах будет колонка, а начиная с планшетов (размер md) — строка. Выравнивание задается классами justify-around, items-center и т.д.

Grid: Для сеток используем grid. Можно очень гибко настроить колонки для разных экранов: grid-cols-1 sm:grid-cols-2 md:grid-cols-3 lg:grid-cols-4. Это создаст сетку, которая автоматически перестраивается с одной колонки до четырех при увеличении ширины экрана. Расстояния между элементами сетки задаются классом gap (например, gap-4).

Анимации и эффекты
В Tailwind есть встроенные анимации:

animate-spin — вращение.

animate-pulse — пульсация.

animate-bounce — прыжки.

Также мы можем комбинировать трансформации: например, при наведении на карточку увеличить её размер — hover:scale-105 — и добавить тень shadow-lg.

Контейнеры и общие стили
Класс container создает адаптивный блок, ширина которого меняется в зависимости от брейкпоинтов. Чтобы отцентрировать его, добавляем mx-auto. Tailwind отлично уживается с обычным CSS. Если вам нужно задать глобальные свойства (например, фон для всего body), вы можете сделать это в файле main.css, используя стандартный синтаксис.
